# 📱 QQ 机器人配置教程

本文档教你如何注册一个 QQ 机器人，让你的 AI 能在 QQ 上和你聊天。

## 准备工作

- 一个常用的 QQ 号（用来登录平台）
- 约 5 分钟时间

## 第一步：登录 QQ 开放平台

1. 打开浏览器访问 https://q.qq.com/
2. 点击右上角「登录」按钮
3. 用你的 QQ 号扫码登录
4. 如果是第一次使用，需要先完善开发者资料

## 第二步：创建机器人应用

1. 登录后，点击「创建机器人」
2. 选择「机器人」类型
3. 填写基本信息：
   - **机器人名称**：给你的机器人起个名字（比如"小助手"）
   - **机器人简介**：简单介绍（比如"一个可爱的 AI 助手"）
   - **机器人头像**：可以上传一张图片，或使用默认头像
4. 点击「提交」，等待几秒钟

## 第三步：获取 AppID 和 ClientSecret

1. 创建成功后，会自动进入机器人管理页面
2. 点击左侧「开发配置」
3. 你会看到两个关键信息：
   - **AppID**：一串数字，比如 `1234567890`
   - **ClientSecret**：一串字母和数字混合
4. **复制这两个信息**，在部署向导中会用到

## 第四步：配置沙箱环境（可选）

QQ 机器人在正式上线前，默认只有你自己能测试：
1. 在机器人管理页面点击「沙箱配置」
2. 添加你的 QQ 号到沙箱用户列表
3. 这样只有你能和机器人聊天（推荐开发阶段使用）

## 第五步：部署工具里填信息

回到部署向导，把刚才复制的：
- **AppID** → 粘贴到「AppID」输入框
- **ClientSecret** → 粘贴到「ClientSecret」输入框

## 常见问题

### ❓ 注册需要审核吗？
创建机器人不需要审核，但如果你想发布到公测/上架，需要平台审核。

### ❓ 机器人能加群吗？
可以，但需要申请开通群聊权限。

### ❓ 每天有消息限制吗？
QQ 开放平台对免费机器人有一定的消息频率限制，正常聊天完全够用。

### ❓ 为什么我给机器人发消息没回复？
可能原因：
1. OpenClaw 还没启动成功
2. 配置的 AppID 或 ClientSecret 不对
3. 你的 QQ 号没添加到沙箱用户列表
4. 机器人还没配置好 Webhook

### ❓ 什么是 Webhook？需要配置吗？
部署向导已经帮你处理好了。如果后续 OpenClaw 换了地址才需要改。

### ❓ 这个收费吗？
QQ 机器人目前是免费的。DeepSeek API 注册送 500 万 tokens，之后按使用量收费（非常便宜）。
